🚀 Demo del Widget ChatBot IA

Esta página es un ejemplo de cómo se vería tu widget embebido en un sitio web real. En la esquina inferior derecha deberías ver la burbuja flotante.

El widget se conecta a tu proyecto Supabase y usa el canal web_widget que configuraste. Los mensajes van directo al motor IA (Claude) si está habilitado en la persona asignada.

Abre el chat, escribe un mensaje y verás que:

  • El mensaje se guarda en tu base de datos
  • Aparece una conversación nueva en tu Inbox en tiempo real
  • Si la IA está activa, Claude responde automáticamente
  • Si el usuario pide hablar con un humano, se detecta el handoff
💡 Para probarlo:
1. Edita este archivo y reemplaza YOUR_CHANNEL_ID, YOUR_SUPABASE_URL y YOUR_ANON_KEY.
2. Abre este HTML en tu navegador (doble clic o sirve con python -m http.server).
3. La burbuja aparece abajo a la derecha.
